## 技术栈

- 用户管理
 - 用户注册与登录
 - 权限控制
- 商品管理

 - 库存管理
- 订单管理
 - 购物车管理
 - 订单创建与处理
 - 订单查询
- 报告与统计
 - 销售报告

- 系统管理员：负责用户管理、系统设置等

- Python 3.8+
- Django 3.2+
- MySQL 5.7+
- 操作系统：兼容Windows、Linux、MacOS
